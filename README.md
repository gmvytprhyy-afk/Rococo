# Roccobot

A Discord bot generated by **BotForge**.

## Features

- Moderation
- Leveling
- Automod
- Logging
- Tickets
- Welcome
- Games
- Utilities
- Message tracking
- Invite tracking
- Economy
- Giveaway
- Web Dashboard

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Copy `.env.example` to `.env` and fill in your credentials:
   ```
   cp .env.example .env
   ```

3. Start the bot:
   ```
   npm start
   ```

## Requirements

- Node.js 18+
- A Discord application with a bot token ([Discord Developer Portal](https://discord.com/developers/applications))
- A Discord OAuth2 redirect URI configured in your app

## Slash Commands

All commands are registered as slash commands. Use `/` to access them.

## Deploy to Railway

This project includes a `railway.json` configuration. Push to GitHub then connect the repo to [Railway](https://railway.app) to deploy instantly.
